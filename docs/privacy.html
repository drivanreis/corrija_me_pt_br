Politica de Privacidade

O `corrija_me_pt_br` foi criado para funcionar de forma local, simples e transparente.

Dados processados

  • A extensao le o texto do campo ativo quando voce pede uma analise ou ativa a verificacao automatica.
  • Esse texto e enviado apenas para o servidor local da propria maquina.
  • O endereco usado e `http://localhost:8081`.

O que a extensao nao faz

  • Nao cria conta.
  • Nao exige login.
  • Nao envia dados para um servidor externo do desenvolvedor.
  • Nao usa rastreamento, analytics ou publicidade.

Armazenamento local

A extensao salva apenas a preferencia de verificacao automatica no armazenamento interno do Chrome.

Controle do usuario

Voce pode desligar a extensao, remover a extensao e limpar os dados locais pelo navegador.
