# Politica de Privacidade do corrija_me_pt_br

O `corrija_me_pt_br` foi criado para funcionar de forma local, simples e transparente.

## Dados processados

- A extensao le o texto digitado no campo ativo quando voce pede uma analise ou quando a verificacao automatica esta ligada.
- Esse texto e enviado apenas para o seu proprio servidor local em `http://localhost`.

## O que a extensao nao faz

- Nao cria conta.
- Nao exige login.
- Nao envia dados para um servidor externo do desenvolvedor.
- Nao vende dados.
- Nao usa rastreamento, analytics ou publicidade.

## Armazenamento local

A extensao salva apenas:

- a preferencia de verificacao automatica

Esses dados ficam no armazenamento da propria extensao no Chrome.

## Controle do usuario

Voce pode:

- desligar a extensao
- remover a extensao
- limpar os dados da extensao pelo Chrome

## Contato

Se voce publicar a extensao, substitua este trecho pelos seus dados reais de contato e pelo link publico do repositório.
