# GitHub Pages

Esta branch existe apenas para publicar a pagina de teste estatica do projeto `corrija_me_pt_br`.

O que vai para o ar aqui:

- `index.html`
- recursos estaticos simples para teste manual da extensao

O que nao vai para o ar aqui:

- backend local
- extensao Chrome completa
- instaladores
- codigo principal da aplicacao

Objetivo:

- oferecer uma pagina publica, simples e controlada
- permitir que o usuario teste a extensao logo depois da instalacao
- manter a aplicacao principal separada na branch `main`
